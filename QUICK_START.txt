🐛 TROUBLESHOOTING:
─────────────────────────────────────────────────────────────────────

Nothing appears?
  → Make sure you opened index.html (not a folder)
  → Try right-click → Open With → Chrome/Firefox

Images not showing?
  → They're loaded from web (Unsplash)
  → Check internet connection
  → Try refreshing (Ctrl+R or Cmd+R)

Search not working?
  → Try search terms: "cardiology", "sarah", "neurology"
  → Check browser console (F12) for errors

Booking won't submit?
  → Fill ALL required fields (marked with *)
  → Date must be in the future
  → Email must have @ symbol
  → Phone must have 10+ digits

Styles look wrong?
  → Clear browser cache (Ctrl+Shift+Delete)
  → Try different browser
  → Make sure styles.css is in same folder
