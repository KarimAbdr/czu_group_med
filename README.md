## Project Overview

MediCare Plus is a fully responsive, accessible medical clinic website created for educational purposes. It showcases best practices in web design, UX/UI, and web accessibility.

## Pages Included

✅ **Homepage** - Hero section, services grid (8+ items with CSS Grid), statistics, and call-to-action
✅ **Our Doctors** - Doctor listing with professional photos and brief descriptions
✅ **Doctor Detail** - Complete doctor profile with education, expertise, and scheduling
✅ **Book Appointment** - Full appointment booking form with validation
✅ **Contact Us** - Contact information, directions, and contact form
✅ **Search Results** - Working search functionality to find doctors and departments
✅ **Responsive Design** - Mobile-first approach for all screen sizes

- **Professional Doctor Photos** - High-quality images from Unsplash
- **SMS Notification Simulation** - Appointment confirmation with SMS message
- **Date/Time Validation** - Prevents booking past dates and times

### ✅ Grid Layout (8+ Elements)
Homepage services grid contains 8 service cards with advanced CSS Grid:
- Card 1 spans 2 columns × 2 rows (featured Emergency Care)
- Card 5 spans 2 columns (featured Pediatrics)

- Fully responsive - adapts to mobile, tablet, desktop
